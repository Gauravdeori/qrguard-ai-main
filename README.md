# QRShield AI

## Project info

A full-stack cybersecurity application called "QRShield AI"

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. **Install dependencies:**
   ```bash
   npm i
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```
